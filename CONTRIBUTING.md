# Contributing to DriveMind

Thank you for your interest in contributing to DriveMind.

DriveMind is a collective memory and intent prediction system for intelligent vehicles.

---

## How to Contribute

You can contribute by:

- Reporting bugs
- Suggesting features
- Improving documentation
- Improving frontend UI
- Improving backend APIs
- Improving AI models
- Adding tests
- Improving Docker support
